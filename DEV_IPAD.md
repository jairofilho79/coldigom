# Testar a aplicação em dev no iPad físico

O **app Flutter (frontend-flutter) foi removido** do projeto. O cliente oficial para desenvolvimento e produção é o **frontend web React** ([frontend/](frontend/)).

Para testar no iPad ou em outros dispositivos, use o frontend React no navegador (responsivo ou PWA, conforme implementado).

Se a pasta `frontend-flutter` ainda existir no repositório (ex.: por permissões em `build/` ou `.dart_tool`), remova-a manualmente na raiz do projeto: `rm -rf frontend-flutter` ou `sudo rm -rf frontend-flutter`.
